# 路径说明

容器内各组件的路径如下：

1. 容器存档启动路径: `/root/.klei/DoNotStarveTogether`
2. 容器存档备份路径: `/app/backup`
3. 容器存档模组路径: `/app/mod`
4. 容器玩家日志路径: `/app/dst-db` （这是一个文件）
5. 容器服务日志路径: `/app/dst-admin-go.log` （这是一个文件）
6. 容器启动饥荒路径: `/app/dst-dedicated-server`
7. 容器启动steamcmd：`/app/steamcmd`
8. first 文件: 代表是否第一次登录，如果没有这个文件会跳转到初始化界面设置账号
